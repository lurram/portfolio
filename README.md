# 🚀 Portfolio | @lurram

Um portfolio moderno, performante e responsivo desenvolvido para destacar habilidades em desenvolvimento Fullstack e Mobile. Este projeto utiliza o que há de mais atual no ecossistema de desenvolvimento web, garantindo uma interface fluida (UI) e uma experiência de usuário (UX) excepcional.

---

## ✨ Funcionalidades Principais

- **🏠 Hero Section Dinâmico:** Apresentação profissional com impacto visual imediato.
- **🛠️ Bento Grid de Habilidades:** Visualização moderna e organizada das competências técnicas.
- **📂 Galeria de Projetos:** Exibição de projetos com categorias (Mobile/Fullstack) e links diretos.
- **🌓 Dark Mode Nativo:** Interface adaptável às preferências do usuário utilizando as novas capacidades do Tailwind CSS 4.
- **✨ Animações Suaves:** Revelação gradual de conteúdo ao scroll para uma navegação engajadora.
- **📱 Responsividade Total:** Design otimizado para todos os tamanhos de tela (Mobile, Tablet e Desktop).
- **📧 Contato Direto:** Integração para cópia de e-mail e redirecionamento para WhatsApp.

---

## 🛠️ Tecnologias de Ponta

Este projeto foi construído utilizando as versões mais recentes das principais tecnologias do mercado:

- **[React 19](https://react.dev/):** A última versão da biblioteca de UI, focada em performance e novas APIs.
- **[Vite 6](https://vitejs.dev/):** Ferramenta de build ultrarápida para uma experiência de desenvolvimento instantânea.
- **[Tailwind CSS 4](https://tailwindcss.com/):** A evolução do framework utility-first, agora mais rápido e com suporte nativo a v4 engine.
- **[TypeScript](https://www.typescriptlang.org/):** Tipagem estática para um código mais seguro e manutenível.
- **[Biome](https://biomejs.dev/):** Ferramenta unificada para linting e formatação de alta performance.
- **[React Icons](https://react-icons.github.io/react-icons/):** Coleção completa de ícones para uma interface rica.

## 🚀 Como Executar o Projeto

Para rodar este projeto localmente, siga os passos abaixo:

### Pré-requisitos
- Node.js (v18 ou superior)
- npm ou yarn

### Instalação
1. Clone o repositório:
```bash
git clone https://github.com/lurram/portfolio.git
```
2. Instale as dependências:
```bash
npm install
```

### Desenvolvimento
Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

### Build para Produção
Gere a versão otimizada para implantação:
```bash
npm run build
```
